## 2. 타이포그래피

- **폰트**: Pretendard (`https://cdn.jsdelivr.net/gh/orioncactus/pretendard@v1.3.9/dist/web/static/pretendard.css`)
- 본문 **15px 이상**, 상품명 **16px 이상** (중장년 사용자 고려)
- 제목 `font-weight:900`, `letter-spacing:-.5~-1px`
- 가격은 크고 굵게 (21~34px, 900)
- 관리자 화면은 버튼·글자 크게 (60대 대표 직접 사용)

## 3. 컴포넌트 규칙

- **카드**: 흰 배경, 1px 보더, `border-radius:12px`, 은은한 섀도(`0 8px 20px rgba(20,27,43,.1)` hover)
- **버튼**: 라운드 `8~10px`. 주요=네이비 솔리드, 보조=네이비 아웃라인, CTA=옐로
- **레이아웃**: 데스크톱 `max-width:1200px` 컨테이너, 모바일 퍼스트 반응형
- **헤더**: 상단 유틸바 → 로고+검색+전화버튼+아이콘 → 네이비 GNB
- **GNB**: [상품▾, 시공사례, 회사소개, 고객센터▾] + 우측 옐로 '시공·견적 문의' CTA
  - '상품' 호버 드롭다운, '고객센터' 호버 드롭다운(공지/문의/자료실)
- **모바일**: GNB 숨김 → 햄버거 드로어. 하단 고정 바 [전화하기/메뉴/내정보/장바구니]
- **전화 상시 노출**: 데스크톱 헤더 옐로 전화버튼, 모바일 하단 고정 바
- **아이콘**: 라인 SVG(stroke 1.8~2) 통일. 이모지 미사용
- **이미지**: 실제 사진 자리는 사선 스트라이프 플레이스홀더 → 실제 이미지로 교체

## 4. 반응형 브레이크포인트

- `max-width:1024px` — 상품 그리드 4열→3열
- `max-width:820px` — 데스크톱 GNB/유틸바/검색 숨김, 모바일 메뉴·드로어·하단바 노출
- `max-width:640px` — 상품 그리드 2열

## 5. 화면별 핵심 요구사항

### 사용자 쇼핑몰

- **메인**: 히어로 슬라이드(3장, 첫 장 h1·나머지 h2), 카테고리 6개, 추천상품 4열, 시공사례 가로스크롤, 공지 3건 + 고객센터
- **상품 리스트**: 좌측 카테고리 사이드바(PC)/상단 칩(모바일), 정렬(최신/낮은가격/높은가격), 품절 오버레이, 페이지네이션
- **상품 상세**: 갤러리+썸네일, 옵션 셀렉트, 수량 스테퍼, 합계. **3상태**: 판매중(장바구니/바로구매) · 품절(버튼 비활성) · 전화문의(가격 대신 `#8F6A00` 텍스트 + 옐로 전화버튼). 상세/배송 탭. 모바일 구매 고정바
- **장바구니/결제**: 체크박스 행·수량·삭제·합계카드 / 3단계·배송지폼·결제수단(무통장 선택 시 입금자명)·약관동의
- **회원**: 로그인 / 회원가입(검증 에러 상태 포함) / 마이페이지(주문내역+상태배지)
- **고객센터**: 시공사례 갤러리+상세 / 공지 테이블(상단 고정 강조) / 문의 폼(유형별 필드 분기, 예약 시 희망일·현장주소, 비밀글, 첨부) / 자료실(다운로드)

### 관리자

- 좌측 다크 네이비 사이드바 + 상단바
- 대시보드: 요약카드 4 + 최근주문 테이블
- 상품관리: 필터 + 테이블 + **판매중↔품절 토글 스위치**(핵심) + 노출여부 + 수정
- 상품등록: 카테고리·상품명·가격(전화문의 체크 시 비활성)·옵션 추가/삭제·이미지 DnD(대표 지정)·상세 에디터
- 주문관리: 상태 탭·검색·상태 셀렉트 / 상세(배송지·상품·상태 플로우·택배사+송장→배송중 처리)
- 문의관리: 미답변 필터·유형 배지·비밀글 아이콘·답변 에디터·답변완료
- 배너관리: 카드 리스트(미리보기·링크·기간·순서 드래그·노출 토글) + 업로드 폼
