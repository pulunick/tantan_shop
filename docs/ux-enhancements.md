# 페이지별 편의 기능 (UX 보강 스펙)

우선순위: [P1] Phase 1 필수 / [P1+] Phase 1 여유 시 / [P2] Phase 2.
구현 시 이 문서와 design-system.md 토큰을 따른다. 디자인 시안은 프롬프트 팩 [9]~[12] 참조.

## 추가 라이브러리 (전부 무료)

| 용도          | 선택                      | 비고                             |
| ------------- | ------------------------- | -------------------------------- |
| 우편번호 검색 | Daum Postcode 스크립트    | 키 발급 불필요                   |
| 이미지 압축   | browser-image-compression | 관리자 업로드 전 클라이언트 압축 |
| 위지윅 에디터 | Tiptap                    | 상품 상세설명/게시글             |
| 토스트        | svelte-sonner             | 전역 피드백 통일                 |
| 라이트박스    | 직접 구현 (native dialog) | 의존성 없이                      |

## 주문/결제

- [P1] Daum 우편번호: zip/addr1 자동 채움 → addr2 포커스 이동
- [P1] 휴대폰 번호 입력 자동 하이픈 (010-0000-0000)
- [P1] 무통장 계좌 복사 버튼 (주문서 + 주문완료 양쪽, 복사 성공 토스트)
- [P1] 주문완료 페이지 = 사실상 결제 화면: 계좌/금액/입금자명 크게, "입금 확인 후 배송 준비" 안내
- [P1+] 입금 기한 문구 (site_settings, 정책은 클라이언트 확인 후)

## 상품 상세

- [P1] 이미지 라이트박스 (확대 + 모바일 핀치줌) — native dialog
- [P1] 전화문의 상품: tel 버튼 + '문의 남기기' 버튼 병행
  → /inquiry?type=quote&product=<상품명> 프리필 (영업시간 외 이탈 방지)
- [P1] 옵션·수량 변경 시 합계 실시간 갱신
- [P1+] URL 복사 버튼 / [P2] 카카오톡 공유, 최근 본 상품

## 장바구니

- [P1] 수량 변경 즉시 합계 재계산, 품절 행 자동 선택 해제 + 안내
- [P2] 무료배송 잔여 금액 바 (배송비 정책 확정 후)

## 상품 리스트

- [P1] 필터/검색/정렬 URL 반영 (뒤로가기·공유 보존 — 기존 설계 확인)
- [P1] 검색 0건: "전화 주시면 찾아드립니다" + 대표번호 (검색 실패 = 전화 기회)
- [P1+] 스켈레톤 로딩

## 문의하기

- [P1] **비회원 문의 조회 /inquiry/check** — 이름+비밀번호로 본인 문의·답변 확인
  (스펙 구멍 보완: RLS상 비회원은 본인 글 조회 불가 → 서버 라우트에서 scrypt 검증 후 표시)
- [P1] 등록 완료 안내: "영업시간(평일 9~18시) 내 연락드립니다"
- [P1+] 첨부 제한 안내 (10MB, 이미지+PDF)

## 회원/마이페이지

- [P1] 비밀번호 보기 토글, 로그인 후 원위치 복귀 (구현됨)
- [P1] 송장번호 → 택배사 조회 링크 (CJ/롯데/한진/우체국/로젠 URL 매핑, 코드 상수)
- [P1] 주문 상태 타임라인 (입금대기→입금확인→배송준비→배송중→완료)
- [P2] 재주문 버튼

## 관리자 (핵심 — 60대 대표 실사용성)

- [P1] 이미지 업로드 전 클라이언트 압축 (최대 1600px / ~300KB)
- [P1] 가격 입력 콤마 자동 표시
- [P1] 주문 목록에서 '입금확인' 원클릭 버튼 (가장 빈번한 작업을 1클릭으로)
- [P1] 위지윅 = Tiptap
- [P1+] 신규 주문/문의 이메일 알림 (Edge Function + Resend 무료 티어) — 클라이언트 회신 후
- [P1+] 배너 권장 사이즈(1920×600) 안내 + 등록 전 미리보기

## 전역

- [P1] svelte-sonner 토스트 (담김/복사/저장 피드백)
- [P1] 이미지 loading="lazy"
- [P1] 404/에러 페이지: 대표번호 + 홈 버튼 (막다른 길 이탈 방지)
- [P1+] 맨 위로 버튼 (모바일)

## 의도적으로 제외 (과투자 판단)

검색 자동완성, 리뷰/별점(P2 확정), 챗봇, 최근 본 상품
