## 타이포

- Pretendard (variable). 본문 15px+, 상품명 16px+, 가격 20~~21px bold, 히어로 34~~44px
- 관리자: 본문 16px+, 버튼 높이 44px+ (60대 사용자)

## 컴포넌트 규칙

- 카드: 흰 배경, 1px #E6E9EE 보더, radius 8~12px, 은은한 섀도
- 버튼: primary=네이비 솔리드/흰 글자, cta=옐로/네이비 글자, outline=네이비 보더
- 품절: 썸네일 반투명 오버레이 + '품절' 배지, 버튼 disabled
- 배지: BEST=옐로/네이비, NEW=네이비/화이트, 11.5px+ bold
- 아이콘: 라인 SVG(stroke #1F3864, 1.8px) 통일. 이모지 사용 금지

## 접근성 체크리스트 (모든 화면 공통)

- [ ] 텍스트 대비 4.5:1 이상 (도구로 검증)
- [ ] 페이지당 h1 1개, 헤딩 순서대로
- [ ] 이미지 alt, 버튼 aria-label
- [ ] 키보드 포커스 스타일 유지, 드롭다운 키보드 접근
